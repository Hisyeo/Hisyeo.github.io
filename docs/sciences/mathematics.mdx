# Mathematics

Mathematics, or sôhoûk, is considered the language of the First Law—the underlying logic of the universe before the Shattering. It is the study of quantity, structure, and change.

## Basic Vocabulary

| English     | Hîsyêô       | Literal Meaning                        |
|-------------|--------------|----------------------------------------|
| Number      | mîkdol       | Amount / Number                        |
| Zero        | xûnyu        | Void / Nothing                         |
| Infinity    | sônsûs       | Endless / Infinite                     |
| Equation    | wofok sôhoûk | Arithmetic proof                       |
| Shape       | xîncûon      | Form / Shape                           |
| Measurement | dûstu        | To measure (Verb) / Measurement (Noun) |

## Operations

Operations are often expressed using prepositions or specific verbs.

| Operation        | Hîsyêô       | Example (2 + 2 = 4)                                                        |
|------------------|--------------|----------------------------------------------------------------------------|
| Plus (Add)       | ûn (and)     | <HisyeoText>dûî ûn dûî li nonkû.</HisyeoText>                              |
| Minus (Subtract) | sûn (from)   | <HisyeoText>dûî sûn nonkû li dûî.</HisyeoText> (Two from four is two).     |
| Times (Multiply) | xokôn (each) | <HisyeoText>dûî xokôn dûî li nonkû.</HisyeoText> (Two each two is four).   |
| Divide           | fun (part)   | <HisyeoText>dûî fun nonkû li dûî.</HisyeoText> (Two parts of four is two). |
| Equals           | li (is)      |                                                                            |

## Algebra

Algebra involves the study of symbols and the rules for manipulating these symbols.

| English  | Hîsyêô           | Literal Meaning           |
|----------|------------------|---------------------------|
| Variable | mîkdol (des fisun) | Unknown number            |
| Constant | mîkdol totsi     | Standing number           |
| Function | ondo sôhoûk      | Arithmetic preparation    |
| Graph    | toswîl sôhoûk    | Arithmetic drawing        |
| To Solve | yosten           | To discover / To find out |

Example Problem:

<HisyeoTabs>
    <p>kûu yosten mîkdol dês fisun Xo.&emsp;<em>Solve for X.</em></p>
    <p>dûî xokôn mîkdol Xo li dus. (zuwobû: tîû)&emsp;<em>2 times X equals 10. (Answer: 5)</em></p>
</HisyeoTabs>

## Geometry

Geometry (xîncûon) deals with the properties of space and the forms that occupy it.

### Fundamental Elements

| English     | Hîsyêô                      | Description                             |
|-------------|-----------------------------|-----------------------------------------|
| Point / Dot | <HisyeoWord is='bindû'/>    | A specific location in space.           |
| Line        | <HisyeoWord is='gîto'/>     | A connection between points.            |
| Edge / Rim  | <HisyeoWord is='dînzoû'/>   | The boundary of a shape.                |
| Straight    | <HisyeoWord is='mûstokîn'/> | Lacking curvature.                      |
| Curved      | <HisyeoWord is='wonkûk'/>   | Bending or deviating from straightness. |

### Describing Polygons

For shapes that do not have a specific name, you can construct a descriptive name using xîncûon (shape) followed by the number of sobô (sides).

Formula:&emsp;<HisyeoWord is='xîncûon'/>&emsp;<HisyeoWord is='yê'/>&emsp;+&emsp;[Number]&emsp;+&emsp;<HisyeoWord is='sobô'/>

| English  | Hîsyêô         | Literal Meaning  |
|----------|----------------|------------------|
| Pentagon | xîncûon yê tîû sobô    | Shape of five sides  |
| Hexagon  | xîncûon yê toû sobô    | Shape of six sides   |
| Octagon  | xîncûon yê tukwos sobô | Shape of eight sides |
| Polygon  | xîncûon yê êto sobô   | Shape of many sides |

<HisyeoTabs>
    <p>xîncûon yê toû sobô fuî toû sobô.&emsp;<em>A hexagon has six sides.</em></p>
</HisyeoTabs>
