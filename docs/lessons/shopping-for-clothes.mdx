# Shopping For Clothes

Whether you are in a marketplace or a supply station, acquiring the things you
need involves specific vocabulary for exchange and description.

## Commerce Vocabulary

In Hîsyêô, commerce is viewed as an exchange of value rather than pure consumption.

<HisyeoTabs></HisyeoTabs>

| English      | Hîsyêô                                 | Literal Meaning     | Usage                                                             |
|--------------|----------------------------------------|---------------------|-------------------------------------------------------------------|
| Store / Shop | <HisyeoText>gubuyo       </HisyeoText> | Store               |                                                                   |
| Market       | <HisyeoText>boxô gubuyo  </HisyeoText> | Store place         |                                                                   |
| Money        | <HisyeoText>munî         </HisyeoText> |                     | Postfix with <HisyeoWord is='xokôn'/> to provide currency amounts |
| Price   | <HisyeoText>wîlûwo       </HisyeoText> | Value / Cost | Not an adjective, needs <HisyeoWord is='fuî'/> to associate with entities     |
| A purchase   | <HisyeoText>kôlmûdul     </HisyeoText> |                     | Not a verb, represents the thing exchanged for                    |
| Cheap        | <HisyeoText>wîlûwo mênûs </HisyeoText> | Deficient cost      |                                                                   |
| Expensive    | <HisyeoText>wîlûwo odigon</HisyeoText> | Excess cost         |                                                                   |
| Taxes        | <HisyeoText>cino         </HisyeoText> | Charge / Fee / Cost |                                                                   |

| To Trade             | <HisyeoText>wedo            </HisyeoText> | Also means "to exchange"                                                            |

## Clothing

The word for "clothing" is &ensp;<HisyeoWord is='ôntôn'/>. It also functions as the verb <em>to wear</em>.

<HisyeoTabs>
    <p>noyo ôntôn solwôc.&emsp;<em>I am wearing a shirt. (Lit: I wear a top one).</em></p>
</HisyeoTabs>

:::tip Standard Verb Object

When you are describing something that you would normally use the verb as the nucleus of a noun phrase, you can skip
the repeated verb, treating the description as having an elided nucleus. If there is confusion, use <HisyeoWord is='ônî'/> as
an [impersonal pronoun](../language/word_types/pronouns#impersonal-pronouns).

<HisyeoTabs>
    <p>noyo ôntôn ônî solwôc.</p>
</HisyeoTabs>

:::

## At the Market

Interactions in a shop are usually direct but polite.

### Asking for Price

To ask how much something costs, you ask what its <HisyeoWord is='wîlûwo'/> (value) is.

<HisyeoTabs>
    <p>¿ôfo fuî wîlûwo zik?&emsp;<em>This has what price?</em></p>
    <p>ni fuî wîlûwo odigon ko?&emsp;<em>Is it expensive?</em></p>
</HisyeoTabs>

### Making the Purchase

<HisyeoTabs>
    <p>noyo fôlun xe wedo til ôfo.&emsp;<em>I want to buy this.</em></p>
    <p>noyo kôî cifû tô munî ko?&emsp;<em>Can I pay with money?</em></p>
</HisyeoTabs>

### Trying it On

If you want to test the fit, you ask to "try" or "test" the clothing using <HisyeoWord is='dênêmêk'/>.

<HisyeoTabs>
    <p>noyo kôî dênêmêk ôntôn ôfo cî?&emsp;<em>I can try this clothing, right?</em></p>
    <p>ni li cûtî odigon.&emsp;<em>It is too small (excessively small).</em></p>
</HisyeoTabs>
