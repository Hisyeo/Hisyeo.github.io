## Talk The Clock

If you know how to count to 12, you can talk about hours; if you know how to
count to 60, then you can talk about minutes and seconds.

To ask the time, you use the copula with a null subject. The complement uses the
interrogative pronoun, so there is no need for a sentence ender.
